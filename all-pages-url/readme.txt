Display and export all page, post, and category URLs in the WordPress admin panel.

== Description ==

All Page URL is a powerful WordPress plugin designed to display all page URLs, post URLs, and category URLs in an organized and easy-to-read table format within your WordPress admin panel. This plugin enhances the user experience by providing a seamless way to view and export URLs for better content management and sharing.

== Installation ==

1. Upload the `all-page-url` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the "Show All URLs" menu item in the WordPress admin panel to view and manage your URLs.

== Frequently Asked Questions ==

= How do I use the export feature? =

Click on the export buttons above the tables to download the URLs in CSV or Excel format.
